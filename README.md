# Economics by Master Ian 📊

A comprehensive A-Level Economics MCQ Quiz platform with 100 unique exam-style questions from 2015-2026.

## Features ✨

- **100 Unique Questions** - No repeats!
- **Years 2015-2026** - Latest exam questions
- **1 Minute Per Question** - Exam-style timing (100 minutes total)
- **Randomized Each Time** - Different order on each attempt
- **Bilingual Explanations** - English & Chinese for every answer
- **Instant Feedback** - Know immediately if you're correct
- **Performance Analytics** - See your score and accuracy

## Topics Covered 📚

- Supply & Demand
- Elasticity (Price, Income, Cross)
- Market Structures (Perfect Competition, Monopoly, Oligopoly)
- Macroeconomics (GDP, Unemployment, Inflation)
- Monetary Policy & Interest Rates
- Fiscal Policy & Taxation
- International Trade & Exchange Rates
- Development Economics
- Market Failure & Externalities
- Consumer & Producer Surplus

## How to Play 🎮

1. Click "START QUIZ"
2. Read the question
3. Select your answer
4. Submit and see feedback
5. Read the explanation
6. Move to next question
7. View final score at the end

## Getting Started 🚀

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
